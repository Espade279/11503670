## Tools Used

| Tool | Website | Description |
|------|--------|-------------|
| Autopsy | https://www.autopsy.com | An open-source digital forensics platform used to analyze disk images and recover artifacts such as browser history, files, and registry data. |
| Registry Explorer | https://ericzimmerman.github.io | A Windows registry analysis tool used to examine user activity, installed programs, and system configuration data. |
| Hex Workshop | https://www.hexworkshop.com | A hex editor used to view and analyze raw binary data within files and disk images. |
